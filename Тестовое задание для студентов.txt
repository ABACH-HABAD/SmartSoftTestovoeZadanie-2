Задание № 2.

Создать соединение с базой данных MySQL.

В базе создать таблицы пользователей и отзывов.

Таблица пользователей - users. 
Будет заполняться из формы "ФИО" (заполнение делать пока не надо).
Поля:
id (int, 11, auto increment, primary key);
name (varchar, 255);
surname (varchar, 255);
email (varchar, 255);
message (text).


Таблица отзывов - reviews. 
Будет заполняться из формы "Информация о заказе" (заполнение делать пока не надо).
Поля:
id (int, 11, auto increment, primary key);
name (varchar, 255);
address (varchar, 255);
comment (text).


В блок отзывы выводить отзывы из базы.
Таблицу reviews заполнить рыбными отзывами. 
Прописать запрос который будет доставать данные из таблицы “reviews” и выводить в блок отзывы. 


Исходники задания и дампы таблиц нужно выложить на github.
